# H-07 如何找到第一个（真实付费）用户

## 一句话定义
**第一个不是你朋友的真实付费用户** = vibe coder 进化成独立开发者的分水岭。它通常**不是从广告 / 海量传播来**，而是从"**精准找到 100 个潜在用户、一对一沟通**"里来。

## 打个比方
**像创业卖手工面包**：你不会先开 100 万粉的抖音，而是**先去小区找 30 个真爱吃面包的邻居**——一个个问"你最喜欢什么口味""愿意花多少钱""怎么改更好"。等他们愿意每周买，你再扩大。

## 和 vibe coding 的关系
- **每个独立开发者最难、最磨人的一步**——从 0 用户到第 1 个付费用户
- 大多数项目死在这一步（不是技术不行，是**没找到愿付费的人**）
- 一旦突破，剩下的就是"找到 10 倍多的同类人"

## 典型场景 / 示例

### 找用户的"7 个最有效渠道"（按 vibe coder 实战排序）

| 渠道 | 转化率 | 成本 | 速度 | 推荐场景 |
|---|---|---|---|---|
| **1. 朋友 / 老同事 / 同学私聊** | 极高 | 零 | 当天 | 第 1-10 个用户 |
| **2. Build in Public（X / 即刻）** | 中 | 零 | 累积式 | 第 10-100 个 |
| **3. Product Hunt（H-01）** | 高 | 零 | 单日爆发 | 第 50-500 个 |
| **4. Reddit r/SideProject / 即刻（H-04 / H-05）** | 中 | 零 | 1-3 天 | 第 10-100 个 |
| **5. 小红书 / 抖音（中文 to C）（H-03）** | 中 | 零 | 累积式 | 中文大众市场 |
| **6. 冷邮件 / 冷私信（B2B）** | 中 | 时间 | 1-2 周 | to B 产品 |
| **7. 付费广告**（Google / Meta） | 低 | $ | 当天 | **不推荐 0→100 阶段用** |

### "10-100-1000 用户" 路径

#### 第 1-10 个用户（10 天内）
1. **列 100 人清单**：你认识的所有可能用你产品的人（朋友 / 同事 / 老同学 / 微信好友）
2. **一对一私聊**："我做了 X，能解决 Y 问题。你愿意试用吗？反馈很重要。"
3. **当天就改产品**：用户提的每条反馈立刻消化，3 天内给反馈
4. **请用户付钱**：哪怕 $1——付钱的反馈比免费的真诚 10 倍

#### 第 10-100 个用户（1-2 个月）
1. **写"我做了 X" 的发布文**——发到 X / 即刻 / V2EX 创意板块 / Reddit / 小红书
2. **每个评论都回复**——评论区互动是 Reddit / 即刻算法的核心信号
3. **PH 发布**（H-01）——准备 7 天，上线日全天值守
4. **在垂直社区发干货**：你的目标用户在哪个 subreddit / 即刻圈子 / 微信群？泡 1 个月

#### 第 100-1000 个用户（3-6 个月）
1. **SEO 教程文**：写"如何用 X 做 Y" 系列博客 / 视频——长尾流量
2. **联盟营销**：给用户分享码——他推一个赚 30%
3. **小红书矩阵 / X thread 矩阵**：成熟内容公式 + A/B 测试
4. **付费广告小测**：单日 $10-$30 测一周，看 CAC（获客成本） vs LTV（用户终身价值）

### 一对一私聊的"提问模板"

打开率 + 回复率最高的 3 种私聊：

#### A. 寻求帮助型
```
嘿 [名字]，
我做了个 [一句话产品]，正想找几个 [目标用户] 试用并给些诚实反馈。
你愿意花 5 分钟看一下吗？不强求买，只想听你最真实的想法。
[链接]
```

#### B. 痛点确认型
```
[名字]，看到你之前在朋友圈/X 抱怨过 [具体痛点]——
我刚做了个工具想解决这个，你愿意试吗？
```

#### C. 直接邀请型（关系熟）
```
[名字]，我新做了一个 [产品]——你是这个领域的高手，
能麻烦你点开看 30 秒、告诉我"这值不值你花 $19 / 月"吗？
```

### 5 条"做错了的信号"

如果你出现下面任何一条，说明**产品 / 定位有问题**，不是营销问题：
1. 你给 50 个朋友看，**只有 1 个想试用**
2. 用户试用了**第二天就不再用**
3. 用户说"挺好"但**不愿付钱**
4. 你写了一段产品介绍，**自己 5 秒说不清"它到底解决什么"**
5. 用户问"我能用 X 替代吗" → 你说"差不多 / 我不知道"

→ 这些信号都指向 PMF（Product-Market Fit）没做到。**先停下宣发，回去改产品**。

## 常见误区
- ❌ **"我建好了就会有人来"**：99% 错。**冷启动需要"主动出击"**——不是被动等流量。
- ❌ **"在小社区刷脸丢人"**：恰相反，**精准小社区的 10 个真实用户 > 大平台 10 万浏览量**。
- ❌ **"PH 是终点"**：PH 只是**第一波种子用户来源**，长期靠产品 + 复购，不靠 PH 单日流量。
- ❌ **"找了 100 个用户没人付费 = 产品不行"**：可能是"沟通话术 / 落地页 / 定价"不对。**反复优化前 100 个用户的转化漏斗**。
- ❌ **"广告能解决冷启动"**：错。**广告只放大已经成立的产品/市场契合**。0 PMF + $1000 广告 = $1000 蒸发。
