# G-11 微信云开发 CloudBase（Tencent CloudBase）

## 一句话定义
CloudBase 是腾讯云的"**一体化 Serverless 后端开发平台**"——云函数 + 云数据库 + 云存储 + Auth 一站搞定，与**微信小程序生态深度绑定**。2026 年活跃运营，主推"AI Coding 后端一体化"（接 Cursor / Claude Code / 通义灵码 / Trae / Qoder 等十余款 AI IDE）。

## 打个比方
**像微信生态里的 Supabase**：把数据库、Auth、Storage、云函数全打包成一个"小程序专用后端套件"，配合 wx-server-sdk 在小程序里 3 行代码调通后端。

## 和 vibe coding 的关系
- **做微信小程序后端最佳选择**（几乎是唯一最优解）
- 2026 年开放给 AI IDE：通过 MCP / CLI / Skills 让 Cursor / Claude Code 直接操作 CloudBase
- 个人版 ¥19.9/月（限时优惠价）对独立开发者很友好

### 「免费体验版 3000 点能跑多少」估算
- 仅跑云函数调用：3000 ÷ 13.3 ≈ **225 万次**（不计资源使用）

- 仅作 CDN 流量：3000 ÷ 210 ≈ **14.3 GB**
- 任一项跑满即耗尽；混用按上面单价加总扣点

### 支持的部署 / 能力
- 微信小程序后端（核心场景）
- 云函数（Node.js / PHP / Python / Go / Java）+ 单实例多并发 + WebSocket
- 云托管（容器，CNB）
- 静态托管
- 云数据库（含 Postgres 关系型）+ 连接自有 MongoDB
- Auth 内置邮件服务
- AI Toolkit (MCP)：接 Cursor / Claude Code / 通义灵码 / Trae / Qoder 等 10+ IDE

### 国内能否直连
- ✅ 极优——国内节点

### 适合场景
- **微信小程序 / 公众号开发**（强绑定，最佳）
- 用 AI IDE 全栈开发，想白菜价开通后端
- 个人项目，预算 ≤¥20/月

### 不太适合
- 跨平台迁移敏感（与腾讯云账号深绑定）
- 重前端展示型 Web 应用（去看 EdgeOne / Vercel）

## 常见误区
- ❌ **"CloudBase 在收缩 / 要下线"**：误传。**2026 年正在活跃扩张**（changelog 几乎每月有重要更新）。
- ❌ **"个人版 ¥19.9 是永久价"**：限时优惠，后续可能恢复原价。
