# C-11 文心快码 Comate

## 一句话定义
文心快码（Comate）是百度官方的 AI 编程工具——2025-2026 期间从单一编辑器插件升级为 **Comate IDE + 多 Agent + Office** 产品矩阵，自称"中国 No.1 的 AI 编程工具"，底层使用百度 ERNIE / 文心系列模型。

## 打个比方
**像百度做的"研发协同套件"**：Comate IDE 是入口，背后是一堆 Agent + 企业研发协同工具（Comate Office）+ Zulu Code Agent。和通义灵码一样面向企业，但产品矩阵更广。

## 和 vibe coding 的关系
- 国内用户、特别是百度生态（智能云、千帆）用户的自然选择
- 完整中文、本地化体验好
- 模型透明度和定价透明度略低（页面未公开具体价格 / 模型清单）

- ❌ **"个人版完全免费无限制"**：有用量限制（具体未公开），重度建议升级。
