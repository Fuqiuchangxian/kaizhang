# C-12 CodeBuddy

## 一句话定义
CodeBuddy 是腾讯云出品的 AI 编程助手，主域名从 copilot.tencent.com 迁至 **codebuddy.cn**，定位"AI 时代的智能编程伙伴"，覆盖 VS Code / JetBrains 插件及独立 IDE 形态。

## 打个比方
**像腾讯版的 Copilot**：和通义灵码 / Comate 一样属于"国内大厂出品的研发助手"，特色是和腾讯云生态打通——CloudBase、混元模型、内部研发流程对接友好。

## 和 vibe coding 的关系
- 腾讯系生态用户的自然选择
- 国内可用，CNY 付款
- 信息披露偏少（官网定价 / 模型清单需访问控制台查看）

### 适合的场景
- 腾讯云生态用户（CloudBase / EdgeOne / 函数计算）
- 企业研发，希望与腾讯云对接
- 中文场景

### 不太适合
- 追求定价 / 模型完全透明（推荐 Trae / Cursor）
- 重度独立开发者 vibe coding（推荐 Trae）

## 常见误区
- ❌ **"copilot.tencent.com 还是主页"**：错。**已迁至 codebuddy.cn**。
- ❌ **"CodeBuddy = 仅腾讯混元"**：底层多模型，但具体清单未公开。
- ❌ **"CodeBuddy = 腾讯云 AI 助手"**：腾讯云有多个 AI 助手产品；CodeBuddy 是其中专做"编程"的那条线。
