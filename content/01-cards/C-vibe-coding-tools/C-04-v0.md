# C-04 v0（Vercel）

## 一句话定义
v0 是 Vercel 旗下的"全栈 Web App AI 生成器"，输入自然语言或截图直接生成 Next.js + React + Tailwind 代码，并能一键部署到 Vercel。**2025-2026 期间主域名已从 v0.dev 切换为 v0.app**。

## 打个比方
**像 Vercel 自家的"AI 流水线"**：你画一张草图、描述一段话，它给你完整一套 Next.js 项目；点一下就上线了。比 Cursor 更"零本地"——你电脑上根本不用装 Node.js / npm；比 Lovable 更"代码可读"——产出的就是标准 Next.js + shadcn/ui 工程，迁移到本地很轻松。

## 和 vibe coding 的关系
- **想做"看起来很漂亮的"Web App / 落地页**：v0 在 React + Tailwind + shadcn/ui 这个具体技术栈上调优得最深，UI 出活儿质量稳定
- **后续要把代码搬回 Cursor 自己改**：v0 的产出文件结构标准、可以无缝迁出
- **已经在 Vercel 部署**：一键链通最丝滑

### 国内能否直接用
- 网站可访问，**国内开发体验受 Vercel CDN 影响，预览页偶有不稳定**
- 部署后默认走 Vercel 全球 CDN，国内访问速度可接受但非最快
- 需国际信用卡付费

### 适合的场景
- 做 Next.js / React 全栈 Web App
- 落地页、产品官网、内部工具
- 已经选好用 Vercel 部署的项目
- 你想"先在 v0 出第一版，再 export 到 Cursor 自己改"的工作流

### 不太适合
- 非 Next.js 技术栈（Vue / Svelte / 原生 HTML）
- 移动 App / 复杂后端
- 完全离线 / 自部署需求

### 2025-2026 关键更新
- **主域名 v0.dev → v0.app**（功能未变，品牌升级）
- 新增 **iOS App**：手机上就能聊天生成
- 新增 **v0 Mini** 廉价模型
- **Design Mode + Design Systems**：组件库化生成
- **Agentic by default**：默认带 Agent，可循环生成

## 常见误区
- ❌ **"v0 = 只能生成单组件"**：早期是的；现在能生成完整全栈项目（含 API 路由、数据库 schema）。
- ❌ **"v0 = 必须用 Vercel 部署"**：可以 export 到 GitHub 然后部署到任何地方。一键 Vercel 只是默认顺手。
- ❌ **"v0 比 Lovable / Bolt 强"**：技术栈强项不同——v0 在 Next.js/React/Tailwind/shadcn 上最稳；Lovable / Bolt 覆盖更广技术栈但深度略浅。
- ❌ **"v0 上的项目可以无限改"**：单项目有 token / credits 上限。复杂项目最好早点 export 到 Cursor 继续改。
