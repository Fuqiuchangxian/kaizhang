# C-15 秒哒 Miaoda

## 一句话定义
秒哒（Miaoda）是百度推出的**无代码 AI 应用搭建平台**，主打"一句话做应用"——生成 H5、小程序、官网、3D 游戏、ERP、问卷、AI 陪伴等。**和 Cursor / Trae 等"生成真实代码"的工具不同，秒哒生成的是托管在百度侧的 App / 小程序**，不属于严格意义的 vibe coding。

## 打个比方
**像"零代码版的 Lovable"**：你说"我要一个能让用户答题、答完显示分数的小程序"，秒哒做出一个能扫码就玩的小程序——但你**不会看到代码、不能下载到本地**。它是产品，不是源码生成器。

## 和 vibe coding 的关系
- **严格意义不属于 vibe coding**：vibe coder 通常需要"可拉到本地、可继续改、可部署到自己服务器"的真实代码
- **但对纯小白 / 营销 / 运营人员极友好**：30 分钟做出个小程序投放微信、抖音
- **作为 MVP 验证工具**有价值：先用秒哒验证用户愿不愿意付费，验证通过再用 Cursor / Lovable 做真正的产品

## 常见误区
- ❌ **"秒哒 = 文心快码"**：错。**文心快码（C-11）是面向开发者的 AI 编程工具**（生成真实代码）；**秒哒是面向非开发者的 no-code 平台**（生成托管应用）。两者都是百度，但场景完全不同。
- ❌ **"秒哒能下载源代码"**：不能（截至 2026-06）。应用绑在百度侧。
- ❌ **"秒哒做的小程序能脱离百度运行"**：默认不能。
- ❌ **"秒哒是 vibe coding 工具"**：严格说不是。vibe coding 强调"通过 AI 写真实代码"——秒哒更接近"AI 驱动的 no-code 平台"。
