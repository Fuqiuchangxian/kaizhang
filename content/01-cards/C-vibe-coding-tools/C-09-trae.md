# C-09 Trae

## 一句话定义
Trae 是字节跳动出品的"AI Coding IDE"——同时提供**国际版（trae.ai）** 和**国内版（trae.cn）**，国内可直接用、可用人民币付款，是 2026 年国内 vibe coder 综合体验最完整的选择之一。**原 MarsCode 已合并进 Trae**。

## 打个比方
**像中国版 Cursor，能在两边买票（CNY / USD）**：界面、Agent 体验对标 Cursor，但配套是字节生态——豆包模型、火山引擎、并且支持本土付款。国内开发者不用翻墙、不用国际信用卡，就能用上专业级 AI IDE。

## 和 vibe coding 的关系
- **国内独立开发者的首推选择**：完整 IDE + 中文界面 + CNY 付款 + 国内网络可用
- 已经合并 MarsCode（原字节的 VS Code 插件线），所以"国际版 IDE + 国内插件" 都在 Trae 旗下
- 也有"TRAE Work"：AI 工作助手 Web / 桌面

### 支持的模型
官方页面未点名清单。公开资料含：豆包系列（自家）、Claude、GPT 等。国内版会自动路由国内可用模型；国际版可接更多海外模型。

### 国内能否直接用
- ✅ **国内版 trae.cn 完全可用**：网络、付款、模型都国内化
- ⚠️ 国际版 trae.ai 国内可访问但部分模型受限

### 2025-2026 关键更新
- **MarsCode 完全合并进 Trae**：trae.cn/plugin 页面明写"原 MarsCode 编程助手"；VS Code 扩展 ID 仍为 `MarsCode.marscode-extension`；旧域名 marscode.cn 自动 301 跳转
- 推出 TRAE Work（AI 工作助手）作为扩展产品线
- 国内版引入"速通"系列档位
- 国际版加入 SOLO Mode

### 适合的场景
- 国内独立开发者，要省心：网络 + 付款 + 模型都国内化
- 不想绑国际信用卡 / 海外网络的用户
- 想用一个 IDE 覆盖大部分 vibe coding 场景

### 不太适合
- 需要最前沿 Anthropic / OpenAI 模型（推荐 Cursor + 国际信用卡）
- 重度英文社区学习路径（社区中文资料更多）

## 常见误区
- ❌ **"Trae 国内版 = 国际版的阉割版"**：功能差异有但不大；很多核心 Agent 能力国内版都有。
- ❌ **"MarsCode 还能单独用"**：MarsCode 品牌已下线，全部收编为 Trae 的插件线。继续从 marscode.cn 进入会自动跳转。
- ❌ **"Trae 只能用豆包"**：豆包是默认，但可切其他模型（要看版本和档位限制）。
- ❌ **"国内版没法用海外模型"**：部分海外模型在国内版仍可调用（具体看档位），但稳定性不如国际版直连。
